# Zʏʀᴏɴɪx Kino Bot

Bu Telegram bot kino ma'lumotlarini taqdim etadi.

## O'rnatish

1. Python 3.11+ o'rnating.
2. Virtual environment yaratish: `python -m venv .venv`
3. Aktivlashtirish: `.venv\Scripts\activate` (Windows)
4. Paketlarni o'rnatish: `pip install -r requirements.txt`

## API Kaliti

OMDB API kalitini olish uchun [omdbapi.com](http://www.omdbapi.com/) ga boring va ro'yxatdan o'ting.

`bot.py` faylida `API_KEY` o'rniga sizning kalitingizni qo'ying.

## Ishga tushirish

`python bot.py`

Bot Telegramda ishlaydi.
